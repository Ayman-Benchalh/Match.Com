# 🛒 Match.Com — Digital Products E-Commerce Platform

A full-featured digital products e-commerce web application built using Laravel, PHP, and MySQL.

The platform allows users to create accounts, browse digital products, place orders, and communicate with the administrator through an integrated messaging system.

---

## 🔗 Project Overview

Match.Com is a Laravel-based e-commerce platform designed for selling digital products such as IPTV subscriptions or online services.

The system includes:

- User authentication
- Product catalog
- Order management
- Messaging system with admin
- Password reset via email

This project demonstrates full-stack Laravel development with database relationships and user interaction features.

## 🚀 Installation

Clone the repository

git clone https://github.com/Ayman-Benchalh/Match.Com.git

Move the project to your server directory (example XAMPP)

xampp/htdocs/

Install dependencies

composer install

Create environment file

cp .env.example .env

Generate application key

php artisan key:generate

Configure database in .env file

Run migrations

php artisan migrate

Start server

php artisan serve

Open in browser

http://localhost:8000
